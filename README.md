Enterprise Campus Network Design with Centralized Control

A scalable enterprise-style campus network design built in Cisco Packet Tracer, featuring VLAN segmentation, redundancy, and centralized security control.

Overview

This project simulates a realistic enterprise campus network — the kind of infrastructure a mid-sized organization would run — with an emphasis on high availability, security, and centralized management.

Key Design Features


VLAN Segmentation — logically separated network traffic by department/function using VLANs
Inter-VLAN Routing — enabled communication between VLANs using Layer-3 Cisco Catalyst switches
Redundancy — implemented RSTP (Rapid Spanning Tree Protocol) and HSRP (Hot Standby Router Protocol) to eliminate single points of failure and ensure high availability
Firewall & Security — configured a Cisco ASA firewall with NAT and ACL-based traffic filtering
Centralized DHCP — automated IP address assignment across the network
Wireless Access Integration — extended connectivity to wireless clients


Testing & Validation


Performed failover testing to confirm redundancy mechanisms (RSTP/HSRP) correctly handle link or device failures
Performed connectivity testing across VLANs and through the firewall to validate secure, reliable network performance


Tools Used


Cisco Packet Tracer — network design and simulation
Cisco IOS — switch and router configuration
Cisco ASA — firewall configuration


Skills Demonstrated

Network architecture design, VLAN/routing configuration, high-availability protocols (RSTP, HSRP), firewall and ACL configuration, network testing and validation.
